# Automated Switch for Clothing Iron Using Wired Motion Sensor

## Overview

This project presents an automated safety system for a clothing iron that reduces the risk of overheating, energy wastage, and accidental fire hazards. The system continuously monitors the movement of the iron using a wired motion sensor and automatically disconnects power when no movement is detected for a predefined period.

## Problem Statement

Conventional electric irons remain powered even when left unattended. This can lead to excessive energy consumption, fabric damage, and potential fire hazards. The objective of this project is to improve user safety by automatically switching off the iron when it remains stationary.

## Features

* Automatic power cutoff during inactivity
* Wired motion sensing mechanism
* Low-cost and reliable design
* Improved electrical safety
* Energy-efficient operation
* Easy integration with existing clothing irons

## System Operation

1. The motion sensor detects movement of the iron during normal usage.
2. As long as motion is detected, power remains connected.
3. If the iron remains stationary for a predefined duration, the control circuit activates a switching device.
4. The switching device disconnects power to the heating element.
5. Power is restored automatically when motion is detected again.

## Components Used

* Motion Sensor
* LM358 Operational Amplifier
* Transistor Driver Circuit
* Relay Module
* Power Supply Circuit
* Passive Components (Resistors and Capacitors)

* LTspice for circuit simulation

## Results

The prototype successfully detected inactivity and disconnected the power supply to the iron after the specified delay period. Experimental testing confirmed reliable operation and improved safety compared to a conventional iron.
